# Project Overview

## Project Name
Open Library Mobile App

## Project Description
A mobile digital library application where users can read public domain books for free and authors can publish their own books under open/public licenses.

The platform focuses on:
- Free access to knowledge
- Open publishing
- Community-driven library ecosystem
- Mobile-first reading experience

## Main Goals
- Allow users to read books for free
- Allow authors to publish books easily
- Support EPUB and PDF reading
- Build a scalable and modern mobile application

## Target Users
### Readers
People who want free legal books.

### Authors
Writers who want to publish open-license books.

### Students
Students looking for educational resources.

## Supported Platforms
- Android
- iOS

## Recommended Tech Stack
### Frontend
Flutter

### Backend
Supabase

### Database
PostgreSQL

### Storage
Supabase Storage

### Authentication
Supabase Auth

## Book Formats
- EPUB
- PDF

## Monetization (Optional Future)
- Donations
- Premium themes
- Audiobook subscriptions
- Author support system

## Licensing
All uploaded books must use:
- Public Domain
- CC0
- CC BY
- CC BY-SA

Users must confirm ownership before publishing.
